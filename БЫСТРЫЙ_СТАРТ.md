# ⚡ Быстрый старт RAG Agent

## 🚀 Запуск за 3 шага:

### 1️⃣ Установка зависимостей
```bash
./install_dependencies.sh
```

### 2️⃣ Запуск всех сервисов  
```bash
./run_all.sh
```

### 3️⃣ Готово! 🎉

**Доступные сервисы:**
- 🔥 RAG API: http://localhost:8000
- 🌐 Web API: http://localhost:5001  
- ⚙️ Admin: http://localhost:5002 (логин: `admin_tg_bot`, пароль: `135beton531`)
- 🤖 Telegram бот: работает в фоне

## ⚠️ Важно:
- Убедитесь, что файл `.env` содержит корректные API ключи
- Первый запуск может занять несколько минут (загрузка моделей)
- Для остановки: `./stop_all.sh` или `Ctrl+C`

Что ожидается от лида компетенций web-разработчика:
Технические навыки :
Глубокое владение фреймворками: React/Vue/Angular (SSR, хуки, контекст).
Понимание паттернов проектирования (HOC, Compound Components).
Оптимизация производительности (Lazy Loading, Code Splitting).
Уверенное использование TypeScript.
Настройка инструментов сборки (Webpack, Vite).
Работа с SSR (Next.js/Nuxt.js), поддержка кроссбраузерности.
Глубокое понимание работы браузера (Event Loop, V8).
Опыт работы с монорепозиториями (Nx, Turborepo).
Менторство и развитие команды :
Наставничество, проведение one-to-one встреч, передача знаний коллегам.
Формирование планов развития для членов команды.
Soft Skills :
Проактивность, работа с заказчиками, объяснение решений команде.
Стратегическое мышление: влияние архитектурных решений на проект.
Управление конфликтами, сглаживание напряжения в команде.
Дополнительные требования :
Понимание влияния своей работы на бизнес-цели проекта.
Умение выявлять и закрывать пробелы в знаниях.
Многозадачность :
Эффективное переключение между задачами/проектами.
Лид должен обеспечивать высокое качество разработки, развивать команду и влиять на стратегические решения проекта.

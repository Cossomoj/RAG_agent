Лид компетенции ожидает от Java-разработчика:
Знание Java : Синтаксис, ООП (наследование, инкапсуляция, полиморфизм), работа с коллекциями (List, Set, Map), исключения (try-catch-finally).
JVM : Понимание работы стека, кучи, сборщика мусора (Garbage Collector).
Spring Framework : Spring Boot, Spring MVC, Spring Data, асинхронное программирование (CompletableFuture, Reactive Streams).
Базы данных : SQL (PostgreSQL, MySQL), Hibernate (Entity, Lazy/Eager loading), NoSQL (MongoDB, Redis), оптимизация запросов.
API и интеграции : REST API, SOAP, gRPC, WebSocket, OAuth2, OpenID Connect, API Gateway.
Инструменты разработки : Maven/Gradle, Docker, Kubernetes, CI/CD (Jenkins, GitLab CI).
Тестирование : Unit-тесты (JUnit), интеграционные тесты (TestNG), автотесты на уровне сервисов.
Микросервисы : Разработка архитектуры микросервисов, Circuit Breaker (Resilience4j, Hystrix), Service Mesh (Istio, Linkerd).
Производительность : Профилирование кода (JProfiler), оптимизация, работа с высоконагруженными системами.
Мониторинг : Настройка мониторинга и алертинга (Prometheus, Grafana).
Soft Skills : Проактивность, работа с заказчиками, объяснение решений команде, выявление и закрытие пробелов в знаниях.
Дополнительно для Senior/Lead : Наставничество, проведение one-to-one встреч, опыт работы с распределенными системами.

Что ожидается от лида компетенций тестирования:
Технические навыки :
Владение стратегиями тестирования (регрессионное, дымовое, интеграционное, системное).
Опыт работы с микросервисной архитектурой и тестированием взаимодействия сервисов.
Понимание работы контейнеров (Docker, Kubernetes) и CI/CD.
Нагрузочное тестирование (анализ узких мест, профилирование).
Работа с API-first подходом, шинами сообщений (Kafka, RabbitMQ).
Глубокие знания SQL/NoSQL баз данных, оптимизация запросов.
Автоматизация тестирования (Selenium, Playwright, Cypress), контрактное тестирование (PACT).
Перехват трафика (Charles, Fiddler).
Менторство и развитие команды :
Наставничество, проведение one-to-one встреч, передача знаний коллегам.
Формирование планов развития для членов команды.
Soft Skills :
Проактивность, работа с заказчиками, объяснение решений команде.
Стратегическое мышление: влияние тестирования на проект, расстановка приоритетов.
Управление конфликтами, сглаживание напряжения в команде.
Дополнительные требования :
Понимание влияния своей работы на бизнес-цели проекта.
Умение выявлять и закрывать пробелы в знаниях.
Многозадачность :
Эффективное переключение между задачами/проектами.
Лид должен обеспечивать высокое качество тестирования, развивать команду и влиять на стратегические решения проекта.

РОЛЬ: Лид компетенций; СПЕЦИАЛЬНОСТЬ: Web разработчик; ТЕМА: Цель встреч один-на-один

Регулярные встречи один-на-один (1-2-1) помогут удержать web разработчика, планировать его развитие таким образом, чтобы это соотносилось со стратегией компании и отвечало меняющемуся бизнес-контексту. Кроме того, это мощный инструмент управления вовлечённостью и доверием сотрудника своему руководителю в области web разработки.

РОЛЬ: Лид компетенций; СПЕЦИАЛЬНОСТЬ: Web разработчик; ТЕМА: Построение доверительных отношений

Такие встречи нужны, чтобы выстроить доверительные отношения лида с web разработчиком. Запланируйте встречу в календаре. Продолжительность встречи — не более 30 минут, регулярность — раз в две недели. Сначала слушай, что расскажет web разработчик о своих проектах и технических задачах, а потом уже спрашивай.

РОЛЬ: Лид компетенций; СПЕЦИАЛЬНОСТЬ: Web разработчик; ТЕМА: Начало встречи

Начни встречу в неформальном стиле, узнай, как дела или что нового и интересного у web разработчика (новые технологии, интересные проекты, технические вызовы), так ты поймешь настроение и готовность к продуктивной беседе. Уточни, есть ли у web разработчика вопросы, которые необходимо обсудить. Если нет, тогда переходи к своим вопросам.

РОЛЬ: Лид компетенций; СПЕЦИАЛЬНОСТЬ: Web разработчик; ТЕМА: Обсуждение целей

Поговорите про личные цели и их выполнение в области web разработки (изучение новых фреймворков, улучшение архитектурных навыков, развитие soft skills). Не забудь поддержать и мотивировать сотрудника, дать обратную связь при необходимости по техническим решениям и профессиональному развитию. Это важно, так ваше общение будет доверительным и честным.

РОЛЬ: Лид компетенций; СПЕЦИАЛЬНОСТЬ: Web разработчик; ТЕМА: Завершение встречи

Проговорите итоги встречи и сроки, если вы что-то ждете друг от друга (выполнение технических задач, изучение новых технологий, участие в проектах, менторство junior разработчиков). Определите конкретные шаги для развития web разработчика и согласуйте план действий на следующий период.

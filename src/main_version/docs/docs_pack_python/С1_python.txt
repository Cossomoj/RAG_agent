Уровень Junior
Soft Skills:
Желание учиться, которое подтверждается делом (что изучено за последний год? Как это применяется?).
Проактивная работа с заказчиком.
Умение принимать обратную связь (ОС).
Многозадачность.
Hard Skills:
Знание синтаксиса Python (типы данных, операторы, условные конструкции, циклы).
Понимание принципов ООП (инкапсуляция, наследование, полиморфизм).
Работа с файлами и базами данных (SQLite, JSON).
Умение работать с библиотеками requests, json, os, logging.
Понимание основ HTTP (методы GET, POST, PUT, DELETE).
Опыт написания простых тестов с использованием unittest или pytest.
Умение развернуть и запустить Python-приложение (виртуальное окружение, зависимости).

Уровень Senior
Soft Skills:
Желание учиться, которое подтверждается делом (что изучено за последний год? Как это применяется?).
Проактивная работа с заказчиком.
Умение принимать ОС.

Опыт работы с микросервисами и монолитами.
Опыт работы с брокерами сообщений (RabbitMQ, Kafka).
Опыт настройки и оптимизации контейнеров в Docker и Kubernetes.
Глубокое понимание многопоточности и асинхронности в Python.
Опыт работы с высоконагруженными системами.
Понимание принципов распределенных систем.
Опыт работы с системами логирования и мониторинга (Grafana, Prometheus).
Опыт работы с протоколами передачи данных (gRPC, WebSockets).
Умение анализировать и устранять узкие места в коде.
Понимание работы LLM и интеграции с внешними AI-системами.
Опыт работы с CI/CD на уровне настройки пайплайнов.
Опыт работы с архитектурой событий (Event-Driven).
Умение программировать на нескольких языках (Python + JS/Go/Rust).

Hard Skills:
Hard Skills Senior и Lead не отличаются.

Многозадачность:
В термин вкладывается возможность переключаться между задачами/проектами (от 2 — оптимально, до 5 — максимально) без сильной потери эффективности.

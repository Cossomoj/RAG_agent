# Матрица компетенций аналитика

## Уровень Junior

### Soft Skills (мягкие навыки):
1. **Желание учиться, подтверждаемое делом**
   - Что изучено за последний год?
   - Как это применяется в работе?

2. **Проактивная работа с заказчиком**
   - Инициативность в коммуникации
   - Предупреждение проблем до их возникновения

3. **Умение принимать обратную связь**
   - Конструктивное восприятие критики
   - Использование ОС для развития

4. **Многозадачность**
   - Переключение между задачами/проектами (2-5 проектов)
   - Сохранение эффективности при переключении

### Hard Skills (технические навыки):
1. **Знание JSON нотации**
   - Понимание структуры данных
   - Умение читать и создавать JSON

2. **Знание Postman/Curl**
   - Тестирование API
   - Работа с HTTP запросами

3. **Умение использовать User Story и Use Case**
   - Формулирование требований
   - Описание пользовательских сценариев

4. **Понимание клиент-серверного взаимодействия**
   - Архитектура веб-приложений
   - Протоколы передачи данных

5. **Владение инструментами разметки макетов**
   - Pixso, Figma или аналоги
   - Создание технических схем

## Определение многозадачности

### Ключевое понятие:
**Многозадачность** - это НЕ возможность одновременно думать о нескольких задачах, а способность эффективно переключаться между задачами и проектами.

### Оптимальные параметры:
- **Минимум**: 2 проекта/задачи
- **Оптимум**: 2-3 проекта
- **Максимум**: 5 проектов

### Критерии эффективности:
- **Сохранение качества работы** при переключении
- **Минимальная потеря эффективности**
- **Способность быстро входить в контекст**
- **Умение приоритизировать задачи**

### Практические рекомендации:
- **Планирование переключений**: Осознанное планирование времени
- **Контекстное переключение**: Сохранение состояния задач
- **Приоритизация**: Четкое понимание важности задач
- **Документирование**: Запись прогресса для быстрого восстановления контекста

### Признаки эффективной многозадачности:
- Быстрое восстановление контекста
- Сохранение качества работы
- Умение балансировать приоритеты
- Эффективное управление временем

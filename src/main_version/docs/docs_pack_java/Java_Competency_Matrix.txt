# Матрица компетенций Java разработчика

## Junior Java Developer

### Базовые технические навыки:
- **Computer Science**: Структуры данных, алгоритмы, gitflow, регулярные выражения
- **Java Core**: Типы данных, коллекции, исключения, строки, ООП, Generics
- **Основы разработки**: Понимание жизненного цикла разработки ПО

### Уровень ответственности:
- Решает простые/комплексные задачи, соответствующие образованию/прошлому опыту
- Действует в рамках зоны своей экспертизы
- Работает под контролем более опытных разработчиков
- Может действовать самостоятельно в рамках поставленной задачи

### Ожидаемые результаты:
- Способность писать рабочий код под руководством
- Понимание базовых принципов разработки
- Умение работать с системой контроля версий
- Готовность к обучению и развитию


РОЛЬ: Specialist; СПЕЦИАЛЬНОСТЬ: Java Developer; ТЕМА: Middle Java Developer - расширенные навыки

## Middle Java Developer

### Дополнительно к уровню Junior:

#### Продвинутые технические навыки:
- **Java Core**: Углубленные знания типов данных, коллекций, исключений, строк, ООП, Generics с опытом применения
- **Дополнительные технологии**:
  - Сериализация объектов
  - Многопоточность и параллельное программирование
  - SQL и работа с базами данных
  - Spring Framework (Core, Boot, Security)
  - JPA/Hibernate для работы с данными
  - Java Performance Tuning
  - Шаблоны проектирования (Design Patterns)

#### Процессные навыки:
- Участие в code review и рефакторинге кода
- Проведение оптимизации производительности
- Понимание жизненного цикла разработки ПО

### Уровень ответственности:
- Действует самостоятельно
- Декомпозирует кросс-функциональные задачи
- Привлекает необходимую экспертизу при необходимости
- Может составлять и приоритизировать бэклог команды совместно с Product Owner


РОЛЬ: Specialist; СПЕЦИАЛЬНОСТЬ: Java Developer; ТЕМА: Senior Java Developer - экспертные навыки

## Senior Java Developer

### Дополнительно к уровню Middle:

#### Экспертные технические навыки:
- **Java Core**: Экспертные знания с глубоким опытом применения
- **Продвинутые технологии**:
  - Продвинутая сериализация и десериализация
  - Сложная многопоточность и синхронизация
  - Оптимизация SQL запросов и схем БД
  - Продвинутое использование Spring экосистемы
  - Сложные JPA маппинги и оптимизация
  - Профилирование и тюнинг производительности
  - Архитектурные паттерны и их применение

#### Лидерские навыки:
- Проведение code review и менторинг младших разработчиков
- Руководство группой разработчиков
- Способность принимать решения в критических ситуациях
- Участие в конференциях и митапах в качестве спикера

### Бизнес-ориентация:
- Решение комплексных кросс-функциональных задач
- Влияние на бизнес-результаты через технические решения
- Участие в развитии экспертизы команды
- Составление и приоритизация бэклога команды совместно с Product Owner


РОЛЬ: Specialist; СПЕЦИАЛЬНОСТЬ: Java Developer; ТЕМА: Lead Java Developer - управленческие навыки

## Lead Java Developer

### Дополнительно к уровню Senior:

#### Управленческие навыки:
- **Развитие экспертизы**: Отвечает за развитие экспертизы или направления
- **Управление командой**: Проявляет способность в управлении командой разработчиков
- **Найм**: Участвует в найме разработчиков
- **Knowledge sharing**: Организует встречи разработчиков по обмену знаниями/опытом

#### Стратегическое мышление:
- Формирование и систематизация задач для развития продукта
- Определение технической стратегии команды
- Принятие архитектурных решений
- Планирование развития технических компетенций

### Ожидаемые результаты:
- Эффективное управление командой разработчиков
- Развитие технической экспертизы в команде
- Влияние на стратегию развития продукта
- Создание культуры постоянного обучения

### Ключевые компетенции:
- Техническое лидерство
- Стратегическое планирование
- Развитие людей
- Управление изменениями
- Коммуникация с бизнесом

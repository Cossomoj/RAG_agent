# Матрица компетенций стажера-аналитика

## Базовые компетенции стажера-аналитика

### Soft Skills (мягкие навыки):
1. **Желание учиться, подтверждаемое делом**
   - Что изучено за последний месяц стажировки?
   - Как новые знания применяются в аналитических задачах?
   - Готовность изучать новые инструменты и методологии

2. **Проактивная работа с ментором**
   - Инициативность в коммуникации с ментором
   - Предупреждение проблем до их возникновения
   - Активное участие в обсуждениях требований

3. **Умение принимать обратную связь**
   - Конструктивное восприятие критики от ментора
   - Использование обратной связи для развития аналитических навыков
   - Готовность корректировать подход к задачам

4. **Базовые навыки многозадачности**
   - Переключение между 2-3 аналитическими задачами
   - Сохранение эффективности при переключении
   - Умение приоритизировать задачи под руководством ментора

### Hard Skills (технические навыки):
1. **Базовое знание JSON нотации**
   - Понимание структуры данных в формате JSON
   - Умение читать и создавать простые JSON документы
   - Применение в анализе API и конфигураций

2. **Начальные навыки работы с Postman/Curl**
   - Базовое тестирование API эндпоинтов
   - Выполнение простых HTTP запросов
   - Анализ ответов серверов

## Анализ требований и документирование для стажера

### Основные аналитические навыки:
3. **Умение использовать User Story и Use Case**
   - Формулирование простых пользовательских историй
   - Описание базовых пользовательских сценариев
   - Понимание структуры: "Как [роль], я хочу [функция], чтобы [ценность]"

4. **Понимание клиент-серверного взаимодействия**
   - Базовое понимание архитектуры веб-приложений
   - Знание основных протоколов передачи данных (HTTP/HTTPS)
   - Понимание роли API в современных приложениях

5. **Начальные навыки работы с инструментами разметки**
   - Базовое владение Pixso, Figma или аналогичными инструментами
   - Создание простых технических схем
   - Аннотирование макетов для разработчиков

### Документирование для стажера:
- **Технические задания**: Создание простых ТЗ под руководством ментора
- **Пользовательские истории**: Формулирование требований в формате User Story
- **Схемы процессов**: Создание базовых диаграмм процессов
- **Аннотации макетов**: Описание функциональности в макетах

### Ожидаемые результаты стажера:
- Способность анализировать простые требования
- Умение документировать базовые процессы
- Понимание принципов создания технических заданий
- Готовность к обучению более сложным аналитическим навыкам

## Коммуникация и командная работа стажера-аналитика

### Коммуникативные навыки:
1. **Эффективная коммуникация с ментором**
   - Регулярные отчеты о прогрессе
   - Задавание уточняющих вопросов
   - Активное участие в one-to-one встречах

2. **Взаимодействие с командой разработки**
   - Участие в обсуждениях требований
   - Предоставление уточнений по ТЗ
   - Сбор обратной связи от разработчиков

3. **Работа с заинтересованными сторонами**
   - Поддержка ментора в сборе требований
   - Участие в интервью с пользователями
   - Документирование результатов встреч

### Командная работа:
- **Коллаборация**: Работа в команде под руководством опытных аналитиков
- **Обмен знаниями**: Делиться изученными материалами с коллегами
- **Поддержка**: Помогать другим стажерам в решении задач
- **Обратная связь**: Предоставлять конструктивную обратную связь

### Развитие навыков:
- **Самообучение**: Активное изучение новых инструментов и методологий
- **Практика**: Применение знаний в реальных проектах
- **Менторинг**: Получение обратной связи от опытных аналитиков
- **Рефлексия**: Анализ собственной работы и поиск путей улучшения

## Путь развития и следующие шаги стажера-аналитика

### Критерии успешного завершения стажировки:
1. **Демонстрация базовых аналитических навыков**
   - Создание качественных технических заданий
   - Эффективный анализ требований
   - Успешная коммуникация с командой

2. **Проактивность и инициативность**
   - Активное участие в проектах
   - Предложение улучшений процессов
   - Самостоятельное изучение новых инструментов

3. **Командная работа**
   - Эффективное взаимодействие с коллегами
   - Поддержка других членов команды
   - Вклад в общие цели проекта

### Следующие шаги развития:
- **Junior Analyst**: Переход на позицию младшего аналитика
- **Расширение навыков**: Изучение более сложных инструментов
- **Специализация**: Фокус на определенной области анализа
- **Менторинг**: Помощь новым стажерам

### Долгосрочные цели:
- **Middle Analyst**: Развитие до уровня среднего аналитика
- **Экспертиза**: Глубокое знание определенных доменов
- **Лидерство**: Управление аналитическими проектами
- **Инновации**: Внедрение новых подходов и инструментов

### Ключевые факторы успеха:
- **Постоянное обучение**: Активное развитие навыков
- **Практический опыт**: Применение знаний в реальных проектах
- **Сетевые связи**: Построение отношений с коллегами
- **Результативность**: Демонстрация измеримых результатов работы

2. Обращается за помощью при затруднениях, не боится признаться в проблеме
3. Проявляет инициативу в коммуникации — уточняет требования, цели, сроки
4. Грамотно воспринимает обратную связь и корректирует поведение/работу
5. Отвечает за результаты своей работы и сдаёт задачи в срок

3. Умение работать с REST API, отправлять запросы, обрабатывать ответы
4. Основы Git: клонирование, коммиты, работа с ветками
5. Понимание принципов адаптивной и кроссбраузерной вёрстки

5. Интересуется продуктом: изучает поведение пользователей, метрики, фидбэк

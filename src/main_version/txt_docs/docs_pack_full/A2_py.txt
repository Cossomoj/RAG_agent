Лид компетенции ожидает от Python-разработчика:
Знание Python : Синтаксис, ООП, структуры данных, асинхронность (async/await), многопоточность (threading, multiprocessing).
Работа с базами данных : SQLite, JSON, PostgreSQL, MySQL, ORM (SQLAlchemy), кэширование (Redis).
API и интеграции : REST API, OAuth, WebSockets, gRPC, работа с внешними API, документирование (OpenAPI/Swagger).
Инструменты и технологии : Docker, Kubernetes, CI/CD (GitHub Actions, GitLab CI), облачные сервисы (AWS, Azure, Cloud.ru).
Тестирование : Unit-тесты (unittest, pytest), интеграционные и нагрузочные тесты.
Архитектура : Проектирование приложений, микросервисы, монолиты, event-driven архитектура.
Оптимизация : Анализ узких мест, работа с высоконагруженными системами, логирование и мониторинг (Grafana, Prometheus).
Системное мышление : Понимание влияния решений на проект, расстановка приоритетов, работа с распределенными системами.
Soft Skills : Проактивность, работа с заказчиками, объяснение решений команде, выявление и закрытие пробелов в знаниях.
Дополнительно для Senior/Lead : Наставничество, проведение one-to-one встреч, опыт работы с LLM и AI-системами, программирование на нескольких языках (Python + JS/Go/Rust).

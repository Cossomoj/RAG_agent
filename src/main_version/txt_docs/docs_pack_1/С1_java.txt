Уровень Junior
Soft Skills: Желание учиться, которое подтверждается делом (что изучено за последний год? Как это применяется?). Проактивная работа с заказчиком. Умение принимать ОС. Многозадачность.
Hard Skills: Знание основ языка Java (синтаксис, примитивные типы данных, операторы, массивы). Понимание основ ООП (наследование, инкапсуляция, полиморфизм). Понимание работы JVM (стек, куча, сборщик мусора). Умение работать с коллекциями (List, Set, Map). Знание основ работы с файлами (java.io, java.nio). Понимание работы исключений (try-catch-finally, throw, throws). Умение подключать внешние библиотеки с использованием Maven/Gradle. Основы работы с REST API (HTTP методы, коды ответа). Умение писать юнит-тесты с использованием JUnit.

Уровень Junior+ / Middle-
Soft Skills: Желание учиться, которое подтверждается делом (что изучено за последний год? Как это применяется?). Проактивная работа с заказчиком. Умение принимать ОС. Многозадачность.
Hard Skills: Продвинутое знание ООП (абстракция, интерфейсы, SOLID-принципы). Понимание потоков и многопоточности (Thread, Executor, синхронизация). Опыт работы с REST и SOAP API (Spring REST, Jersey). Опыт работы с базами данных (SQL, Hibernate). Умение писать интеграционные тесты (JUnit, TestNG). Понимание работы с логированием (Log4j, SLF4J). Опыт работы с Docker (запуск контейнеров). Умение работать с Git (rebase, merge, cherry-pick). Понимание устройства HTTP-протокола.

Уровень Middle+
Soft Skills:
Желание учиться, которое подтверждается делом (что изучено за последний год? Как это применяется?). Проактивная работа с заказчиком. Умение принимать ОС. Умение доносить свои мысли до коллег. Умение объяснить заказчику возможные варианты реализации. Многозадачность. Умение выявить у себя недостаток знаний в определенном домене и закрыть его при необходимости.
Hard Skills: Свободное владение Spring Framework (Spring Boot, Spring MVC, Spring Data). Понимание асинхронного программирования (CompletableFuture, Reactive Streams). Опыт работы с очередями (RabbitMQ, Kafka). Глубокое понимание работы Hibernate (Entity, Lazy/Eager loading, кэширование). Опыт написания автотестов на уровне сервиса и интеграции. Опыт работы с микросервисной архитектурой (разбиение, взаимодействие). Опыт конфигурирования Docker и Docker Compose. Понимание работы с контейнерами в Kubernetes. Опыт работы с API Gateway. Понимание принципов работы транзакций (ACID). Опыт работы с NoSQL базами (MongoDB, Redis). Умение оптимизировать запросы к базе данных. Понимание работы сервисов в stateless/stateful режиме. Умение разрабатывать RESTful и gRPC API.

Уровень Senior
Soft Skills: Желание учиться, которое подтверждается делом (что изучено за последний год? Как это применяется?). Проактивная работа с заказчиком. Умение принимать ОС. Умение доносить свои мысли до коллег. Умение объяснить заказчику возможные варианты реализации. Многозадачность. Умение выявить у себя недостаток знаний в определенном домене и закрыть его при необходимости. Понимание, как работа влияет на проект в целом: что нужно сделать в первом приоритете, что можно поставить на паузу, чего можно не делать вообще. Умение сглаживать напряжение внутри команды, умение объяснить команде, что могут быть задачи интересные, но не полезные для проекта.
Hard Skills: Опыт построения архитектуры микросервисов. Понимание работы с OAuth2, OpenID Connect. Опыт работы с Circuit Breaker (Resilience4j, Hystrix). Свободное владение Kubernetes (поднятие кластера, деплой). Опыт работы с API-first подходом. Опыт работы с gRPC и WebSocket. Понимание плюсов и минусов синхронного и асинхронного взаимодействия. Умение профилировать и оптимизировать код (JProfiler). Опыт настройки и работы с Service Mesh (Istio, Linkerd). Умение настраивать CI/CD пайплайны (Jenkins, GitLab CI). Опыт работы с реляционными и NoSQL базами (PostgreSQL, MongoDB, Redis). Свободное знание принципов работы Garbage Collector в JVM. Опыт настройки мониторинга и алертинга (Prometheus, Grafana). Опыт работы с распределенными системами.

Уровень Lead
Soft Skills: Как у Senior +: 10. Наставничество над коллегами из своей компетенции с понятным результатом — приобретением ими желаемых скиллов. 11. Умение давать ученикам нетравматичную ОС. 12. Умение проведения встреч one-2-one. 
Hard Skills: Hard Skills Senior и Lead не отличаются.

Многозадачность:
В термин вкладывается возможность переключаться между задачами/проектами (от 2 — оптимально, до 5 — максимально) без сильной потери эффективности.
